claude·tape · 手机端

多 tab 形态 · 三选一

都基于现有设计语言（浅色 Apple 风 + 珊瑚品牌色）。点开任一方案即可在手机框里实际点击体验：切换会话、新建、关闭、进配置/外网。手机上直接打开则铺满全屏。

A
浏览器式切换器
顶栏只放当前会话名 + 「⊞ N」按钮；点开全屏卡片网格（带终端缩略图），点卡片切换。底部一排去 主页/配置/外网。
最省顶栏空间会话多也不挤手机最熟悉切换多一步点击
›
B
横向滚动 tab 条
顶栏下钉一条可左右滑的 tab 药丸条，最忠实移植桌面 tab-strip。Home 钉最左、会话带状态点 + ×、最右是「＋」。
与桌面端一致tab 始终可见一点即切会话多需横滑
›
C
底部导航 + 滑动会话
底部固定导航（主页/会话/新建/配置/外网）；会话之间左右滑动切换，顶部圆点指示。导航与会话彻底分离。
纯拇指操作导航最清晰滑动切会话不够直观底部占一行
›
提示：在方案 A 里点 ⊞ 试切换器；方案 B 点药丸/＋/× 试 tab 条；方案 C 点终端左右两侧 ‹ › 模拟滑动切会话。选定后我再按它正式重构 mobile.html / mobile.js（会话并发模型、外网/云端等功能补齐再单独定）。
